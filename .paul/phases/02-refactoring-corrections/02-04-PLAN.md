  <name>Task 2: Validation speed + retrait admin</name>
  <files>web/hardware/views.py, web/driftapp_web/urls.py, tests/test_web_views.py</files>
  <action>
    1. **H-19 — Validation speed dans GotoView et JogView :**
       Dans les deux views, remplacer le bloc `if speed is not None: params['speed'] = float(speed)` par :
       ```python
       if speed is not None:
           try:
               speed = float(speed)
           except (TypeError, ValueError):
               return Response(
                   {'error': 'Vitesse invalide'},
                   status=status.HTTP_400_BAD_REQUEST
               )
           if not (0.00001 <= speed <= 0.01):
               return Response(
                   {'error': f'Vitesse hors limites (0.00001-0.01): {speed}'},
                   status=status.HTTP_400_BAD_REQUEST
               )
           params['speed'] = speed
       ```

    2. **L-21 — Retirer admin :**
       Dans `urls.py`, supprimer `from django.contrib import admin` et `path('admin/', admin.site.urls),`
