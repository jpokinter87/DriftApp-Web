  <action>
    Lancer un audit de code complet sur core/ en utilisant les agents d'exploration.

    Pour chaque module, examiner :
    1. **Bugs latents** : conditions de bord, erreurs silencieuses, race conditions
    2. **Sécurité** : injection, validation d'entrées, gestion d'erreurs trop large
    3. **Qualité** : code mort, duplication, nommage incohérent, complexité excessive
    4. **Performance** : opérations coûteuses, allocations inutiles, I/O bloquant
    5. **Maintenabilité** : couplage fort, responsabilités mixtes, absence de types
