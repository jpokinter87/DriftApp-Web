<task type="auto">
  <name>Task 1: Créer GitHub Actions CI</name>
  <files>.github/workflows/ci.yml</files>
  <action>
    Créer `.github/workflows/ci.yml` :

    ```yaml
    name: CI
    on:
      push:
        branches: [main, 'v5.0/*']
      pull_request:
        branches: [main]

    jobs:
      test:
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v4
          - uses: astral-sh/setup-uv@v4
          - name: Install dependencies
            run: uv sync --dev
          - name: Lint
            run: uv run ruff check core/ services/ web/
          - name: Tests
            run: uv run pytest tests/ --tb=short
          - name: Coverage
            run: uv run pytest tests/ --cov=core --cov=services --cov-report=term-missing
    ```

    Adapter si nécessaire : variable DRIFTAPP_DEBUG=1 pour les tests Django.
  </action>
  <verify>Le fichier YAML est syntaxiquement valide : `python3 -c "import yaml; yaml.safe_load(open('.github/workflows/ci.yml'))"`</verify>
  <done>AC-1 satisfait</done>
</task>
