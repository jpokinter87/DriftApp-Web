# Project: DriftApp

## Description
Application de pilotage de coupole astronomique pour l'Observatoire Ubik. DriftApp positionne et maintient automatiquement le cimier (trappe) de la coupole aligné avec le télescope durant les sessions d'astrophotographie. Elle s'appuie sur une abaque empirique (~300 mesures) pour interpoler la position de coupole en fonction de l'azimut et de l'altitude de l'astre visé, et gère le suivi continu avec différents régimes moteur adaptés à la criticité de la phase.

## Core Value
L'astrophotographe indique l'astre visé et la coupole se positionne puis suit automatiquement le mouvement apparent du ciel, maintenant le cimier aligné avec le télescope — y compris lors du retournement de méridien.
